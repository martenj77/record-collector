Kom igång med git (och github)!
<!--more-->
## Starta ett projekt

### Klona ett befintligt

```PS
git clone https://github.com/krompaco/record-collector.git
```

### Forka ett befintligt
```
git fork https://github.com/krompaco/record-collector.git
```

### Starta från noll
```
md new_project
cd new_project
git init
```
